<summary>⚙️ <b>Snake not showing? Click here — One-Time Setup Guide</b></summary>
<br/>

**Step 1:** Create the file `.github/workflows/snake.yml` in your profile repo (`gagan-k454545/gagan-k454545`):

```yaml
name: Generate Contribution Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

permissions:
  contents: write

jobs:
  build:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - name: Generate snake.svg
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - name: Push snake.svg to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**Step 2:** Go to **Actions tab → "Generate Contribution Snake" → Run workflow**

**Step 3:** Wait ~1 minute for the snake to generate in the `output` branch. Done! 🎉
